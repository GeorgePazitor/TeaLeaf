# TeaLeaf
TeaLeaf implementation in C++ with OpenMP and MPI support based on the reference implementation of TeaLeaf from [UK-MAC/TeaLeaf_ref](https://github.com/UK-MAC/TeaLeaf_ref).

## Compiling

Using cmake :

```
mkdir build
cd build
cmake ..
cmake --build .
```

## Usage

To use the existing file "tea.in" :

```
./tealeaf
```
